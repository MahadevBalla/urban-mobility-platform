# Audit Notes: Urban Mobility Platform

## Focus: data fusion, zone generation, and evaluation validity

This directory documents a review of the system, with emphasis on the **data fusion** and **zone generation** pipelines.

The intent is to record issues that affect:

- correctness of outputs
- validity of experimental conclusions
- reproducibility
- future extensibility

The findings are split into:

- **[System-level issues](./SYSTEM_ISSUES.md)** that affect multiple modules or invalidate assumptions.
- **[Module-level issues](./MODULE_ISSUES.md)** tied to specific files and implementations.
